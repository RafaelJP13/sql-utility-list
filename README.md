# SQL Utility List

A collection of SQL and MySQL examples created to help beginners learn the fundamentals of relational databases.

> ⚠️ **Legacy Project**
>
> This repository was originally written as a tutorial for an online forum and was later migrated to GitHub for preservation.
>
> It is one of my earliest projects, created while I was still learning SQL. Although it does not represent my current coding standards or best practices, I decided to keep it public as part of my learning journey.

---

## 📚 Contents

This repository includes examples of:

- SQL Command Dictionary
- Entity Relationship Model (ER Model)
- Database Creation
- Data Insertion
- Table Alteration
- Data Selection
- Aggregate Functions
- Joins
- Stored Procedures

## 🎯 Purpose

The goal of this project is to provide a simple reference for people who are beginning to learn SQL and MySQL.

Many of the examples are intentionally straightforward so that newcomers can understand the syntax before moving on to more advanced concepts.
